# Tarokka Deck

*Draws a random Tarokka card from a specified deck*

# Version
v1.0.0

# Examples
Basic usage:
`!tarokka`
`!tarokka high`
`!tarokka common`

Available options:
- `full`: Draw from the full Tarokka deck.
- `high`: Draw from the High Deck.
- `common`: Draw from the Common Deck.

# Limitations
- Draws one card at a time.
